    Steps to Activate Intel® Active Management Technology or Intel® AMT¶

    1. Connect a device to a Monitor and power up.

    2. The Intel® AMT platform displays the BIOS startup screen during power up, then processes the BIOS Extensions.

    Entry into the Intel® AMT BIOS Extension (MEBx) is BIOS vendor-dependent. If you are using an Intel® AMT reference platform (SDS or SDP), the display screen prompts you to press Ctrl+P. Then, the control passes to the Intel® Converged Security and Management Engine (Intel® CSME) or MEBx main menu.

    1. Enter the Intel® CSME default password admin.

    2. Change the default password (required to proceed).

    3. From the Main Menu, select Intel® AMT Configuration.

    4. Manageability Feature Selection: Select/Verify this is set to - Enabled.

    5. Activate Network Access: Select this option. Then, select Y to confirm AMT activation.

    6. Network Setup. Select this option. Then, select Intel ME Network Name Settings. Enter Host Name (Not a mandatory step).

    7. SOL/Storage Redirection/KVM: Select this option. Enable all the required features - Username and Password, SOL, Storage Redirection and KVM Feature Selection.

    8. User Consent: By default, this is set for KVM only; Change this to NONE.

    9. Save and Exit Intel® CSME (or MEBx).

    10. When the AMT device platform or operating system opens, connect to the internet, via an ethernet cable (wifi support not available).

    11. Download the MeshCMD suitable to your device platform from the Meshcommander.com site.

    Note: Document or remember the download location of meshcmd.exe as you will need to run this later to get the GUID.

    Add new device

    For an OEM system, it is easy to use the one-time boot menu, although entry into Intel® CSME is usually an included option as part of the one-time boot menu. The exact key sequence varies by OEM, BIOS and Model.

    New passwords must be a strong. These should contain a minimum of the following:

    • 8 characters
    • one Uppercase letter
    • one lowercase letter
    • one digit
    • one special character
